# PROJECT NEON

Projekt Neon ist ein 2,5D Sidescroller, bei dem man auf einem Hoverboard verschiedene Level überwinden muss.
Das Spiel wurde mit der Unreal Engine Version 5.3 entwickelt. Zum Spielen werden Maus und Tastatur benötigt.

## Systemanforderungen
- Microsoft Visual C++ Umgebung
- Nvidia Graphics Driver Verson 527.27 (vergleichbar/besser)
- 1 GB Speicherplatz

## Bedienung
A/D: Bewegung nach links/rechts
Space: Sprung und wenn noch in der Luft, dann Doppelsprung
S: Crouchen
L-Shift: Dash in aktuelle Bewegungsrichtung
E: Pausenmenü
